# 🐟 RPU App - PT Rafasya Putra Ustanto

Aplikasi pencatatan penjualan ikan untuk PT Rafasya Putra Ustanto.

## 📱 Fitur
- ✅ Input transaksi batch (multiple transaction)
- ✅ Rekap data penjualan
- ✅ Rekap per bongkaran
- ✅ Master data (Pembeli, Ikan, Bongkaran)
- ✅ Cetak laporan
- ✅ PWA - Install ke Android

## 🚀 Cara Install

### 1. Update URL Apps Script
Ganti `SCRIPT_URL` di `script.js` dengan URL deployment Anda:
